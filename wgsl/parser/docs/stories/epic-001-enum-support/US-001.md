# 🎯 US-001: Ajouter le lexing du mot-clé `enum`

**Epic Parent** : [EPIC-001: Support Complet des Types Énumérés](../../WGSL_PARSER_COMPLETION_EPICS_AND_STORIES.md#-epic-001-support-complet-des-types-énumérés)

---

## 📋 Description

**En tant que** : Développeur utilisant le parser WGSL  
**Je veux** : Que le lexer reconnaisse `enum` comme un mot-clé  
**Afin de** : Pouvoir déclencher le parsing approprié pour les déclarations d'enum

---

## ✅ Critères d'Acceptation

- [ ] `enum` est ajouté à `TokenKind` comme un keyword
- [ ] `enum` est reconnu dans `Lexer.keywordFor()`
- [ ] Le token `ENUM` est généré quand `enum` apparaît dans le source
- [ ] Test unitaire: `Lexer("enum MyEnum {}").tokenizeSignificant()` contient `TokenKind.ENUM`
- [ ] Test unitaire: `TokenKind.ENUM.isKeyword == true`

## 🎯 Exemple de Code

### Avant (non-conforme)
```kotlin
// TokenKind.kt
enum class TokenKind {
    // ... autres tokens
    STRUCT,
    ALIAS,
    // enum manquant
}

// Lexer.kt
private fun keywordFor(text: String): TokenKind? = when (text) {
    "struct" -> TokenKind.STRUCT
    "alias" -> TokenKind.ALIAS
    // "enum" manquant
    else -> null
}
```

### Après (conforme)
```kotlin
// TokenKind.kt
enum class TokenKind {
    // ... autres tokens
    STRUCT,
    ALIAS,
    ENUM,  // ✅ Ajouté
}

// Lexer.kt
private fun keywordFor(text: String): TokenKind? = when (text) {
    "struct" -> TokenKind.STRUCT
    "alias" -> TokenKind.ALIAS
    "enum" -> TokenKind.ENUM  // ✅ Ajouté
    else -> null
}

// TokenKind.kt - isKeyword
val TokenKind.isKeyword: Boolean
    get() = when (this) {
        // ... autres keywords
        TokenKind.ENUM -> true  // ✅ Ajouté
        else -> false
    }
```

## 🧪 Tests Attendus

```kotlin
// LexerEnumTest.kt
class LexerEnumTest : FunSpec({
    test("enum keyword is recognized") {
        val tokens = Lexer("enum MyEnum {}").tokenizeSignificant()
        tokens shouldContain Token(ENUM, span, "enum")
    }
    
    test("ENUM is a keyword") {
        TokenKind.ENUM.isKeyword shouldBe true
    }
    
    test("enum in declaration context") {
        val tokens = Lexer("enum Color { RED, GREEN }").tokenizeSignificant()
        tokens[0].kind shouldBe TokenKind.ENUM
    }
})
```
